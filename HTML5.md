
# Anatomy of an HTML document
----

```html
<!DOCTYPE html>
<html lang="en-US">
	<head>
		<meta charset="utf-8" />
		<title>My test page</title>
	</head>
	<body>
		<p>This is my page</
```

# Void Elements
----
Self closing tags. Such as 
```html
<img src="#" alt="#" />
```
These insert / embed something in the document
