## How to Mass Import

### Section 1: Mass Import Users

- 1.1
    
    In your web browser visit: [https://portal.azure.com/#home](https://portal.azure.com/#home)
    
- 1.2
    
    In the Search Tab, look for Active AD (Azure Active Directory)
    
- 1.3
    
    Click on the Bulk Operations, Click on Download
    

![](https://images2.imgbox.com/00/4b/4ldObI8H_o.png)

- 1.4
    
    Fill out the Name, Username, Password, Block sign up, First name and last name, etc of the CSV file
    

![](https://images2.imgbox.com/7d/f4/TyDB5yGl_o.png)

- 1.5
    
    Upload the CSV file from the folder you saved it in (E.g. Downloads folder, etc)
    
- 1.6
    
    Click the Submit button
    

![](https://images2.imgbox.com/10/aa/Yi2Zql1P_o.png)

- 1.7
    
    The list of users on the CSV file should be imported on your Azure AD Users List
    

![](https://images2.imgbox.com/b6/41/IZneWvfP_o.png)
