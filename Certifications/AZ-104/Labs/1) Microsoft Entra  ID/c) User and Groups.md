## How to create/delete Users and Groups

### Section 1: Create Groups

- 1.1
    
    In your web browser visit: [https://portal.azure.com/#home](https://portal.azure.com/#home)
    
- 1.2
    
    In the Search Tab, look for Active AD (Azure Active Directory)
    
- 1.3
    
    Click on the Groups Tab
    

![](https://images2.imgbox.com/65/13/g0E0y6kJ_o.png)

- 1.4
    
    Click on New Group
    

![](https://images2.imgbox.com/aa/c5/BUDyTIxT_o.png)

- 1.5
    
    Select Security Type, Enter in Group name, Group description, and select the membership type (Assigned in this example), etc
    
- 1.6
    
    Click on Create
    

![](https://images2.imgbox.com/64/83/o5Dpr09i_o.png)

### Section 2: Create Users

- 2.1
    
    Click on the Users Tab
    

![](https://images2.imgbox.com/ee/02/jFxA7Dp7_o.png)

- 2.2
    
    Click on New User
    

![](https://images2.imgbox.com/74/0b/YXHIvMtV_o.png)

- 2.3
    
    Enter in the User name, Name, First name, Last name, and select the Password type
    

![](https://images2.imgbox.com/d1/e9/OUjVR1de_o.png)

- 2.4
    
    To assign the user to a group(s), click on **0 groups selected**
    
- 2.5
    
    Select the Group desired (In this example: Developers), then Press Select
    

![](https://images2.imgbox.com/23/1d/CdIexmVL_o.png)

- 2.6
    
    Click on User
    
- 2.7
    
    Select roles desired for the user and click on Select
    
- 2.8
    
    Click on Create
    

![](https://images2.imgbox.com/5f/12/BsWxqBGi_o.png)

- 2.9
    
    The new user should be created, you can also delete and restore users within a 30-day time frame.
    

![](https://images2.imgbox.com/36/43/gGN3o3DU_o.png)
