## Launching an ARM Template

### Section 1: Launching an ARM Template

- 1.1
    
    In your web browser visit: [https://portal.azure.com/#home](https://portal.azure.com/#home)
    
- 1.2
    
    In the Search Tab, type for deploy
    
- 1.3
    
    Click on Deploy a custom template
    

![](https://images2.imgbox.com/40/db/lCfQIEwm_o.png)

- 1.4
    
    Click on Build your own template in the editor
    

![](https://images2.imgbox.com/80/d6/fAz7qwqv_o.png)

- 1.5
    
    Click on Add Resource
    
- 1.6
    
    Select the Ubunto virtual machine as shown in the demo (Or whichever resource you want to create)
    
- 1.7
    
    Enter in the details and Click OK
    

![](https://images2.imgbox.com/a5/08/Ube4IGnS_o.png)

- 1.8
    
    Click Save
    

![](https://images2.imgbox.com/c5/5c/dzMIFZtT_o.png)

Reference: [Microsoft.Compute virtualMachines](https://docs.microsoft.com/en-us/azure/templates/microsoft.compute/virtualmachines?tabs=json)

- 1.9
    
    Select Subscription, the resource group, and enter in parameters, etc
    
- 1.10
    
    Click on Review + Create
    

![](https://images2.imgbox.com/94/5e/xMTuoZLa_o.png)

- 1.11
    
    The deployment is completed and the virtual machine should be added to your resource group list of resources
    
- 1.12
    
    Click on Go to Resource Group
    

![](https://images2.imgbox.com/c1/42/ocsR3rCn_o.png)

### Section 2: Deleting resources, resource groups

- 2.1
    
    You can delete resources individually or in a group. You can delete the entire resource group by clicking Worf (Name of the resource group shown in the demo)
    
- 2.2
    
    Click on Overview
    
- 2.3
    
    Click on Delete Resource group
    
- 2.4
    
    Confirm and click Delete
    

![](https://images2.imgbox.com/6a/a9/uYSeAlUQ_o.png)

### Section 3: Exporting templates

- 3.1
    
    You can export your template to reuse or share with others by selecting the resources you want
    
- 3.2
    
    Click on Export Template
    
- 3.3
    
    Click on Download and you should have a saved file of it ready to be used
    

![](https://images2.imgbox.com/83/21/53eUkceJ_o.png)

### Reference

[Microsoft.Compute virtualMachines](https://docs.microsoft.com/en-us/azure/templates/microsoft.compute/virtualmachines?tabs=json)
