## Create an Azure Container Instances

**Section 1 - Create an Azure Container Instances**

-   1.1
    
    Search for **Container Instances** in the search tab, Click on +Create to create a Container Instance
    
    ✋
    

Enter in the information

-   Create a new Resource Group: banana
-   Name: banana (or whichever is available)
-   Region: East US 2
-   Image source: Quickstart images
-   Image: mcr.microsoft.com/azuredocs/aci-helloworld:latest (Linux)

Click Review+Create

-   1.2
    
    Browse Overview, Settings: Containers, Identity, Properties, etc and get familiar with Azure Container Instances. Copy the IP address (Public) to check the "Welcome to Azure Container Instances!" instance as well!
    
    ✋
    
-   1.3
    
    When you are done exploring, delete the resource group.
