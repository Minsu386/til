### Returning Boolean values
----

```JavaScript
function isEqual(a,b) {
  if ( a === b ) {
    return true;
  } else {
    return false
  }
}
```
There is a better way to do the above. Since === returns true or false we can simplify to:
```JavaScript
function isEqual(a, b) {
  return a === b;
}
```
